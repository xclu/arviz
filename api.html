API Reference¶

Plots¶

autocorrplot(posterior[, var_names, …]) Bar plot of the autocorrelation function for a posterior.
compareplot(comp_df[, insample_dev, …]) Summary plot for model comparison.
densityplot(data[, data_labels, var_names, …]) Generates KDE plots for continuous variables and histograms for discretes ones.
energyplot(trace[, kind, bfmi, figsize, …]) Plot energy transition distribution and marginal energy distribution in order to diagnose poor exploration by HMC algorithms.
forestplot(data[, kind, model_names, …]) Forest plot
kdeplot(values[, cumulative, rug, label, …]) 1D KDE plot taking into account boundary conditions
parallelplot(trace[, varnames, figsize, …]) A parallel coordinates plot showing posterior points with and without divergences
posteriorplot(data[, var_names, coords, …]) Plot Posterior densities in the style of John K.
traceplot(data[, var_names, coords, …]) Plot samples histograms and values.
pairplot(trace[, varnames, figsize, …]) Plot a scatter or hexbin matrix of the sampled parameters.
jointplot(data[, var_names, coords, …]) Plot a scatter or hexbin of two variables with their respective marginals distributions.
khatplot(khats[, figsize, textsize, ax, …]) Plot Pareto tail indices.
ppcplot(data, ppc_sample[, kind, mean, …]) Plot for Posterior Predictive Checks

Stats¶

bfmi(trace) Calculate the estimated Bayesian fraction of missing information (BFMI).
compare(model_dict[, ic, method, b_samples, …]) Compare models based on the widely applicable information criterion (WAIC) or leave-one-out (LOO) cross-validation.
hpd(x[, credible_interval, transform, circular]) Calculate highest posterior density (HPD) of array for given credible_interval.
loo(trace, model[, pointwise, reff]) Pareto-smoothed importance sampling leave-one-out cross-validation
r2_score(y_true, y_pred[, round_to]) R² for Bayesian regression models.
summary(trace[, varnames, round_to, …]) Create a data frame with summary statistics.
waic(trace, model[, pointwise]) Calculate the widely available information criterion, its standard error and the effective number of parameters of the samples in trace from model.
psislw(log_weights[, reff]) Pareto smoothed importance sampling (PSIS).

Diagnostics¶

effective_n(trace[, varnames, round_to]) Returns estimate of the effective sample size of a set of traces.
gelman_rubin(trace[, varnames, round_to]) Returns estimate of R for a set of traces.
geweke(trace[, varnames, first, last, intervals]) Return z-scores for convergence diagnostics.
